guake
=====


What is it
----------

Guake is a top-down terminal for Gnome (in the style of Yakuake for KDE, Tilda
or the terminal used in Quake).

You can check the screenshots to get a small feeling about it.


How to get it
-------------

Guake is packaged by a number of distributions, among which are Fedora, Debian, 
Ubuntu or ArchLinux, so please check your repo. Source tarballs are available;
see the `Downloads page <http://guake.org/downloads>`_.


Reporting bugs
--------------

Oh the bugs. Report 'em all! To file a bug report, use `github ticketing system
<https://github.com/Guake/guake/issues>`_.
